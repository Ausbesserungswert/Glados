# Glados
Glados from the game Portal with lamp function

# Credits
All credits are going to Yvo de Haas for his great work documented [here](https://ytec3d.com/glados-lamp/)

We thank you and hope to contribute to this project.

# Abstract
Even though the original project was fun to build, it is missing some details about the electronics and controller. This contribution will provide a complete solution to bring Glados to life.
